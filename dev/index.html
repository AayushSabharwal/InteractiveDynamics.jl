InteractiveChaos is a package that provides applications for interactively exploring dynamical systems. It extends various packages of JuliaDynamics.

To install it do ]add InteractiveChaos Makie. Makie is necessary for providing a plotting backend, since InteractiveChaos does not install one by default.

The functionality of InteractiveChaos is contained within individual functions, all of which launch a dedicated interactive application. Here is their list:

  • interactive_orbitdiagram
  • interactive_poincaresos
  • trajectory_highlighter
  • interactive_billiard
Example animations

Besides the documentation strings, each interactive function is accompanied with an animation (.gif or .mp4 file) displayed after the docstring! All of these animations are generated by simply running examples from the examples folder. Some of these applications are potentially used in the JuliaDynamics YouTube channel
